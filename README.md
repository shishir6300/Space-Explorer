# 🌌 Space Explorer

> **An interactive solar system explorer built with vanilla HTML, CSS & JavaScript — powered by NASA's live API.**

🔗 **Live Demo:** [shishir6300.github.io/space-explorer](https://shishir6300.github.io/space-explorer)

---

## ✨ Features

- 🪐 **All 8 Planets** — Click any planet for detailed stats, fun facts & more
- 🔍 **Planet Search** — Filter planets instantly by name or type
- 📡 **Live NASA APOD** — Today's Astronomy Picture of the Day fetched in real-time
- 🚀 **Historic Missions** — Apollo, Voyager, Mangalyaan, JWST and more
- 📰 **Space Facts Ticker** — Scrolling fun facts about the universe
- 🌟 **Animated Star Field** — Dynamic hero with orbiting sun animation
- 📱 **Fully Responsive** — Works beautifully on mobile, tablet and desktop
- ✉️ **Contact Form** — Clean form with success feedback

## 🚀 Getting Started

### Run Locally
```bash
# Clone the repo
git clone https://github.com/shishir6300/space-explorer.git

# Open in browser
cd space-explorer
# Open index.html with Live Server in VS Code
```

### Setup NASA API Key
1. Get your **free** API key at [api.nasa.gov](https://api.nasa.gov)
2. Open `script.js`
3. Replace the API key:
```javascript
const API_KEY = 'YOUR_NASA_API_KEY_HERE';
```
